# Question: Screen Sourcing

What display fits the Mondivox 705's unusual ultra-wide aperture (≈10.5 × 3 cm)?

## Facts
- Available space: ~10.5 × 3 cm total; usable control width ~7.5 cm.
- Standard displays don't match this aspect ratio.
- Leads found:
  - Youritech 2.7" bar-style displays: https://www.youritech.com/products/2-7-inch-bar-style-displays/
  - Youritech 4" OLCD 960×160 MIPI AMOLED with capacitive touch: https://www.youritech.com/products/4-inch-olcd-display-960x160-mipi-amoled-display-with-capacitive-touch.html
  - Multi-display with I2C multiplexer: complex, not preferred.
- Alternative: keep the original frequency-range backlight as a fixed display (no active screen).
- Prototype-first approach suggested: validate software before committing to screen choice.

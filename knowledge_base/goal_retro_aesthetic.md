# Goal: Retro-Futurist Aesthetic

The device should look and feel like a vintage radio that has been quietly upgraded — "retro-futurism 3.0".

## Facts
- The vintage shell is a strong preference, not a hard requirement — a custom-built case is an accepted alternative.
- A vintage host radio (e.g. Mondivox 705) is the current starting point, not a locked choice.
- Any new parts (screen, 3D-printed brackets) should not clash with the vintage aesthetic.
- Adding a generic modern screen risks destroying the look — screen is optional (see [[question_screen_sourcing]]).

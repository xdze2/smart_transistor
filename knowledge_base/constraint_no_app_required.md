# Constraint: No Phone or App Required for Normal Use

Day-to-day operation (power, volume, channel selection) must work without a smartphone or companion app.

## Facts
- Wi-Fi setup and streaming-service credential entry are accepted as one-time admin exceptions.
- All runtime controls must map to physical hardware on the device.

# Smart Transistor — Design Documents

This directory contains the design output for the Smart Transistor project: a vintage transistor radio (Mondivox 705) retrofitted into a WiFi streaming music player with physical controls and no phone required.

## What these docs are

Structured engineering output, synthesized from the knowledge base. They record *what we decided* and *why*, in a form that can be handed to a builder (ourselves, later).

They are not a blog. They are not brainstorm notes. Those live in `internal_docs_raw/` and `knowledge_base/`.

## Design approach

We follow a lightweight, iterative process (see [[meta_engineering_design_process]]): no formal gate reviews, decisions recorded as they are made, open questions kept explicit rather than papered over.

The knowledge base (`knowledge_base/`) is the living reference. These design docs are the *output layer* — they freeze decisions and form the basis for building.

## Project in one sentence

A vintage transistor radio with its original knobs and aesthetic intact, streaming internet radio (and maybe Spotify/Deezer), booting in under 5 seconds, needing no phone after initial Wi-Fi setup.

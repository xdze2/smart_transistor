# Goal: Retro-Futurist Aesthetic

The device should look and feel like a vintage radio that has been quietly upgraded — "retro-futurism 3.0".

## Facts
- The host hardware (case, knobs, screen bezel) should be preserved as much as possible.
- Any new parts (screen, 3D-printed brackets) should not clash with the vintage aesthetic.
- Adding a generic modern screen risks destroying the 1940s/1950s look.

# Constraint: Stateless Controls

Each physical control does exactly one thing — no mode-switching, no multi-function buttons.

## Facts
- "One knob does only one thing" — stated as TBD/desired in brainstorm.
- Conflicts with limited control surface: the Mondivox 705 has a fixed number of knobs.
- Required controls: hard on/off, volume, channel selection.
- Optional: shuffle, play/pause, skip next/back.
- Explicitly not needed: search, playlist management, fast-forward/rewind, time selection.

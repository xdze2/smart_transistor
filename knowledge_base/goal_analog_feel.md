# Goal: Analog Feel

The device should feel like a physical, analog appliance — not a computer with a screen and menus.

## Facts
- Controls are stateless: each knob/button does exactly one thing.
- No smartphone required for normal use.
- The original haptic interfaces (knobs, sliders, toggles) of the host radio are preserved or replaced with equivalents of similar quality.
- **Sleep mode is an acceptable substitute for instant cold boot.** The device is mains-powered and always indoors; leaving it in sleep between uses is fine. What matters is that it *feels* always-ready, not that it boots in 3s from cold.
- **Controls must be self-explanatory** — a guest should be able to use it without explanation. No mode-switching, no hidden functions, no need to read a manual.
